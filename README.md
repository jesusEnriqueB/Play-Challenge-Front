# Play-Challenge-Front

Use Node v20.10.0

# Instructions to run project:
- npm i
- Set env field, NEXT_PUBLIC_API_URL after initialize front project (default: http://localhost:3006)
# start proyect (method 1)
- npm run dev
# start proyect (method 2) 
- npm run build
- npm run start
